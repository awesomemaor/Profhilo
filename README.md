# 3D Developer Portfolio - Maor | Software Engineer

A professional, interactive 3D portfolio website built to showcase my journey as a Software Engineer graduate from Braude College of Engineering. This project features immersive 3D models, smooth animations, and a fully functional contact system.

## 🚀 Live Demo

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite, Tailwind CSS
* **3D Rendering:** Three.js, React Three Fiber, React Three Drei
* **Animations:** Framer Motion
* **Email Service:** EmailJS
* **Deployment:** Vercel

## ✨ Key Features
* **Interactive 3D Models:** Dynamic 3D assets (including a custom-mapped avatar and Earth model).
* **Responsive Design:** Optimized for all screen sizes, from mobile to ultra-wide monitors.
* **Functional Contact Form:** Directly sends emails to my inbox using EmailJS API.
* **Professional Experience:** Detailed timeline of my academic projects and military service.
